Como funciona

• As notificações funcionam mesmo com o app fechado, desde que o Vyn esteja instalado como PWA no seu celular.

• Os alertas são gerados quando você abre o app e agendados para o horário configurado (8h–9h).

• Para receber notificações no celular, adicione o Vyn à tela inicial primeiro.

iPhone: Safari → Compartilhar → Adicionar à Tela de Início

Android: Chrome → ⋮ → Adicionar à tela inicial
